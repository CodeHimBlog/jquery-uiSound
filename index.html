jQuery UI Sound Effects Library with uiSound Plugin

uiSound let's you easily play a variety of UI sound effects on different events including typing, click, hover, slide and much more...

Written in: HTML5 & JavaScript
Framework: jQuery 3.3.1

Fork on GiHub Download

Hover Sound Effect 

$(".sample-hover").on("mouseenter", function(){

  $(this).uiSound({play: "hover"});

});

Browser Compatibility

Best result in Google Chrome and Firefox. Older browser are not supported.

How to use

You can trigger sound effect on any event (including custom events) and in any jQuery function. There is the same method to initialize the uiSound plugin inside the various types of functions.
The method is: 

$(this).uiSound({play: "sound-name"});

The following is the list of built in sounds, however you can also add your custom sounds.
